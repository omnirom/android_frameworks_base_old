public int getStatusCode ()

Retrieve the status code of the submit score operation. This will be one of the following values:

  • STATUS_OK if the operation succeeded.
  • STATUS_NETWORK_ERROR_OPERATION_DEFERRED if the operation could not be completed at this time. The operation will be cached and retried again later.
  • STATUS_NETWORK_ERROR_OPERATION_FAILED if the operation encountered a non-recoverable network error. In this case, the operation will not be retried.

Returns
  • The status code of the submit score operation.
